**Why:** Menggantikan proses manual/kertas menjadi alur digital terpusat.

**How to apply:** System ini untuk sivitas akademika UNESA. Brand: Tegas, Transparan, Terpusat. Anti-references: SaaS korporat generik (biru-putih template), portal pemerintah lama. A11y target: WCAG AA.

Register impeccable: `product` (app UI serving a task, not brand marketing).
